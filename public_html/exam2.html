4. Three part question

  1. Saturation is how bold and intense the color but mainly it is how the color looks under a certain lighting and is related to chromaticity, which is how "shiny" a color looks.
  2. Hue is purely the color or what you would simply refer to the color as without taking into consideration any other modifications. (Such as blue, green, red, etc.)
  3. Value is how light or dark the color is and can be compared to that of the brightness/contrast function on a computer screen (notice the color changes).
